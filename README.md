# HuggingFace LLM Course — Notebooks

Hands-on implementations of the HuggingFace LLM Course chapters.

## Contents
- `GithubIssuesSemanticSearch/` — Semantic search over GitHub issues using FAISS and Sentence Transformers
- `TokenClassification/` — NER fine-tuning with BERT on CoNLL-2003
- `Fine-tuning a masked language model/` — Fine-tuning DistilBERT on the IMDB dataset for Masked Language Modeling (MLM) using Accelerate
